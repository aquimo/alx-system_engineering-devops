# Shell, permissions

Project done during **0x01. Shell, permissions**. It aims to learn about man pages, permissions (owner, group and other) of files and directories in **Shell**.

## Technologies
* Scripts written in Bash 4.3.11(1)
* Tested on Ubuntu 14.04 LTS
